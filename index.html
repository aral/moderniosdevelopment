This three‐day intensive workshop written and presented by experience designer and developer Aral Balkan will teach you solid foundations in designing and developing native iPhone and iPad apps using modern practices in Objective-C and Cocoa Touch. The course covers the latest tools, technologies, and devices, including Xcode 4.5, Objective-C 2.0, iOS6, and the new iPhone 5.

You will spend three days hands‐on coding with Aral as you learn how to build native apps that conform to Apple’s coding practices and human interface guidelines.

The aim of the class is not to razzle dazzle you by showing you all of the latest snazzy APIs. Instead, you will be empowered with a solid foundation in Objective‐C and Cocoa Touch so that you understand the culture and conventions of the iOS platform and can subsequently learn to use any of the thousands of iOS APIs you may encounter in your own projects.

Highlights

The following topics, and more, will be covered in this intensive, three‐day workshop:

  • Learn about design considerations for the entire iOS platform, including hardware differences between devices (including the new iPhone 5 and new iPad) and user experience considerations when designing your apps.
  • Write professional code that is consistent with the conventions of Cocoa Touch and the system frameworks using Xcode 4.5 and iOS6.
  • Get a solid foundation in Objective-C and and Cocoa Touch
  • Learn about classes, protocols, categories, methods, and properties
  • Learn important patterns like Model View Controller, Observer, Mixin, etc., and how they are implemented and used in Cocoa Touch with data sources, target/action pairs, delegates, notifications, and more.
  • Learn modern practices, including using the latest collection literals (literals for arrays, dictionaries, etc.) and modern property syntax (with full understanding of what properties are shorthand for)
  • Working with Automatical Reference Counting (ARC)
  • Background on older methods of working (manual memory management, etc.) so you can work with legacy code.
  • Using class extensions to implement ‘private’ methods and properties
  • Using blocks for asynchronous tasks.
  • Asynchronous networking, consuming APIs, and using JSON
  • Creating hybrid apps that use HTML, CSS, and JavaScript via the UIWebView
  • Optimising your workflow for high resolution, including the 4" display of the new iPhone 5
  • Social integration with Twitter and Facebook via the Social Framework
  • Discussion of important iOS6‐specific features like Collection Views, UI State Preservation, Auto Layout, and Data Privacy.
